### TYPESCRIPT 

### Introduction

Typescript is a sub-set of Javascript. It is a developer tool for assigning types to data/input. Javascript is dynamic data typed while typescript is static typed.

We will look into the following:

Srings
Numbers
Boolean
Functions
Objects
Array
Tuple
Enum
Aliases
Literals
Union
Void
